# Etapa de geração de relatórios
projeto de automação RPA.

## Finalidade
Apartir de um relatorio no formato CSV deve-se ser gerado 3 relatorios agrupando:

- Enviados
- Criticados
- Finalizados

## versao python
3.10

## Dependências
- Faker
- python-dotenv

### Instalar dependências
```shell
pip3 install Faker python-dotenv 
```

## Como rodar

No diretório workers rode o comando abaixo:
```shell
python3 app.py
```
